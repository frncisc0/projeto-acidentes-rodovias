## Descrição
Este projeto tem como objetivo realizar uma análise exploratória dos registros de acidentes em rodovias federais brasileiras no período de 2021 a 2025.

## Dataset
Os dados foram obtidos no portal de dados abertos da Polícia Rodoviária Federal (PRF).
O dataset consolidado possui aproximadamente 342 mil registros e cerca de 30 variáveis.

## Estrutura do Projeto

- data/: arquivos CSV dos acidentes
- scripts/: scripts de carregamento e análise dos dados
- docs/: documentação do projeto

## Tecnologias utilizadas

- Python (pandas)
- R (RStudio)
- GitHub

## Etapa 1

A Etapa 1 do projeto contempla a definição do tema, escolha do dataset, organização do repositório e planejamento inicial da análise.

O documento completo da etapa está disponível na pasta `/docs`.

## Etapa 2

A Etapa 2 do projeto contemplou a Análise Exploratória de Dados (AED), utilizando Python para tratamento, organização e visualização das informações obtidas no dataset da Polícia Rodoviária Federal (PRF).

Foram desenvolvidas análises relacionadas:
- à evolução dos acidentes ao longo dos anos;
- aos estados com maior número de ocorrências;
- à gravidade dos acidentes;
- aos tipos de acidentes mais frequentes;
- às condições meteorológicas;
- às fases do dia com maior concentração de acidentes.

Os scripts desenvolvidos encontram-se na pasta `/scripts`.

Os gráficos e resultados obtidos foram utilizados como base para interpretação dos padrões identificados nos dados.

---

## Etapa 3

A Etapa 3 do projeto teve como foco a construção do Data Storytelling, buscando transformar os dados analisados em uma narrativa visual e interpretativa.

Nesta etapa foram integrados:
- gráficos quantitativos e categóricos;
- interpretação dos insights obtidos;
- narrativa baseada em personagens fictícios;
- contextualização dos dados analisados;
- preparação do roteiro para futura apresentação do projeto.

O storytelling foi desenvolvido utilizando os personagens fictícios Sebastião (caminhoneiro) e Maria (guarda rodoviária), permitindo aproximar os dados analisados da realidade vivenciada nas rodovias federais brasileiras.

O documento completo da etapa está disponível na pasta `/docs`.
